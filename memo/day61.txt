0. 복습
1) DOM(Document Object Model)
   웹 문서를 하나의 객체로 만든 것이다
   객체이기 때문에 여러 프로퍼티와 메소드를 제공한다
   DOM을 이용하면 웹문서에 존재하는 모든 HTML요소를 자바스크립트에서 객체로
   가져와 사용할 수 있다

2) DOM 트리
   DOM은 웹 문서의 구조와 정보들을 트리형태로 저장한다

         document
             |
            HTML
         HEAD   BODY
          META  TITLE  H1      A
            tree-test    href  네이버
            
   DOM트리를 구성하는 하나하나의 요소를 노드라고 하고 최상위 노드를 root 노드라고 한다
   DOM트리의 최상위 노드는 document 노드이고, 각 노드들은 위치에 따라 부모, 자식, 형제, 조상 등의
   노드로 불리기도 한다
   태그들만 노드로 존재하는 것이 아닌 태그들의 속성이나 컨텐츠(내용)들도 해당 태그의 자식노드로 존재한다
   속성은 어트리뷰트 노드, 텍스트는 텍스트 노드라고 부른다

3) DOM 사용하기
   document.메소드()
   DOM에서는 HTML 요소를 선택자를 이용해 접근하는 여러 메소드를 제공한다

4) 요소내용 가져오기, 수정하기
   요소.innerText
   요소.innerHTML

5) 요소 속성 가져오기, 수정하기, 삭제하기
   요소.getAttribute('속성명')
   요소.setAttribute('속성명', '값')
   요소.removeAttribute('속성명')

6) DOM을 활용한 이벤트 처리
   - DOM요소에 이벤트 핸들러를 직접 연결시키기
      요소.onXXX = 함수;
   - addEvenetListener() 메소드를 사용하기
      요소.addEventListener(이벤트, 함수);

1. 라이트박스
   라이트 박스는 여러 사이트에서 사용하는 효과이다
   작은 이미지를 클릭하면 큰 사진으로 보여주는 등 여러상황에서 활용할 수 있다.

1) data속성
   data-데이터이름 = '데이터'
   브라우저는 무시하는 속성이므로 특별한 기능은 없으며 개발자가 필요한 데이터를 저장하는 속성이다
   자바스크립트에서 data속성을 가져와서 사용할 수 있다.

2) data 속성 사용방법
   요소.getAttribute('data-데이터이름')
   요소.dataset.속성명

   ex) let test = document.querysSelector('.btn').dataset.test;
   console.log(test);

2. 팝업창과 모달창
1) 팝업창
   형재 페이지에서 새로운 브라우저 창을 열어주는 것
   브라우저 설정에 따라 팝업창을 막아두는 경우도 있다
   최근에는 사용 빈도가 급격히 떨어졌다

2) 모달창
   라이트박스를 마치 하나의 팝업창처럼 사용한다
   팝업창과 비교했을 때 실제 브라우저 창이 아니라 새로운 레이어를 띄워준다는 차이가 있다
