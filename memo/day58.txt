1. 생성자 함수와 클래스
- 생성자 함수
   함수 선언을 통해 생성자를 정의한다
   생성자 함수 내부에서 this 키워드를 사용하여 객체의 프로퍼티를 초기화한다
   new 키워드를 이용하여 생성자 함수를 호출하여 객체를 생성한다

- 클래스
   class 키워드를 사용하여 클래스를 정의한다
   클래스 내부에 constructor 메소드를 정의하여 객체의 프로퍼티를 초기화한다
   new 키워드를 이용하여 클래스를 호출하여 객체를 생성한다

2. 배열
- 배열 리터럴 대괄호([])를 대괄호를 사용하는 방법
   let arr = [];  //빈배열 생성
   arr[0] = 1;
   arr[1] = '1';   

+) 자바에서 빈배열 생성
   자료형[] 배열명 = {};
   자료형[] 배열명 = new 자료형[칸수];   

- 초기값 할당
   let ar1 = [1, 2, 3];

- 배열 크기 지정(쉼표 개수만큼 크기가 지정)
   let ar2 = [ , , , , ];

- Array() 생성자 함수를 사용하는 방법
   let ar3 = new Array();

- 초기값 할당
   let ar4 = new Array(1, 2, 3, 4);

- 배열 크기 지정
   let ar5 = new Array(3);
   //요소가 1개이고 숫자인 경우 배열 크기로 사용된다

3. 배열에서 사용하는 반복문
1)  for ... in
  객체의 열거 가능한 속성을 순회할 때 주로 사용
  배열이나 객체의 key값 순회
  배열에서는 인덱스가 키로 사용되므로 배열의 인덱스를 순회하게된다

2) for ... of
  이터러블 객체 (반복가능한 객체, 배열/문자열/map/set 등)를 순회할 떄 주로사용
  value(배열의값)을 순회
  배열의 각 요소 (값)을 순서대로 순회한다

4. map() 함수
  배열의 각 요소에 대해 주어진 콜백함수를 호출하고 각 요소를 새로운 배열로 매핑시킨다

  const newArray = array.map(callback(element[, index[, array]][, thisArg]));

  element : 현재 처리중인 요소의 값
  index : 현재 처리중인 요소의 인덱스(선택)
  array : map()를 호출한 배열(선택)
  thisArg : callback 함수 내에서 this 키워드가 참조할 객체(선택)

5. filter() 함수
   주어진 콜백함수로 모든 배열요소를 테스트하고 조건을 충족하는 요소만을 가지고 새로운 배열을 생성한다

   const newArray = array.filter(callback(element[, index[, array]][, thisArg]));

   element : 현재 처리중인 요소의 값
   index : 현재 처리중인 요소의 인덱스(선택)
   array : filter()를 호출한 배열(선택)
   thisArg : callback 함수 내에서 this 키워드가 참조할 객체(선택)
